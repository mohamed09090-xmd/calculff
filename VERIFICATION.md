## حالة أدوات البيئة

| الأمر | الحالة | التفاصيل |
|---|---|---|
| `flutter pub get` | لم يُنفذ | Flutter SDK غير مثبت، والبيئة تمنع تنزيله من الإنترنت. |
| `flutter analyze` | لم يُنفذ | يتطلب Flutter SDK. |
| `flutter test` | لم يُنفذ | يتطلب Flutter SDK وحزم المشروع. |
| `flutter build apk --debug` | لم يُنفذ | Flutter وAndroid SDK غير متوفرين في بيئة البناء الحالية. |

لا تُعرض الأوامر السابقة على أنها ناجحة؛ لم يكن تشغيلها ممكنًا تقنيًا في هذه البيئة.

## تحقق خارجي جاهز

الملف `.github/workflows/build-apk.yml` يشغّل تلقائيًا على GitHub Actions:

```bash
flutter pub get
flutter analyze
flutter test
flutter build apk --debug
```

ويحفظ APK الناتج كـ artifact باسم `game-credit-profit-manager-debug-apk`.
